# OriginalDocuments

The original source documents (the `.docx`/`.xlsx`/`.htm` files that the
Markdown protocols were converted from) are **not** stored in this repository.

They are retained in the original drafting repository, where the
`docx_to_markdown.py` conversion workflow was developed:

- <https://github.com/ArdenB/APPN-Aerial-Standard-Operating-Procedures/tree/main/OriginalDocuments>

The canonical, maintained versions of every protocol now live as Markdown under
[`Protocols/`](../Protocols) in this repository. Edit those files directly —
the original source documents are kept only as a historical archive.
